## 💡 핵심 확정 기능
1. **하이브리드 대시보드:** 혈당(메인)과 혈압(서브)을 한눈에 보는 직관적 UI.
2. **멀티 차트 분석:** 혈당/혈압 각각의 기간별 추이, 의학적 가이드라인(기준선/최고최저선) 완벽 제공.
3. **통합 진단 엔진:** 앱 전체에 동일하게 적용되는 무결성 의학 알고리즘.

## 📁 lib 디렉토리 파일 구조 및 기능

### 1. 진입점 및 설정 (`lib/`)
- **`main.dart`**: 애플리케이션 진입점. Flutter 엔진, `.env` 환경 변수, 카카오 SDK, 알림 서비스 및 Firebase 초기화를 담당하며, 전역 테마(라이트/다크 모드) 전환 상태를 관리합니다.

### 2. 모델 및 상태 로직 (`lib/models/`)
- **`health_stat.dart`**: `HealthRecord` 데이터 모델 정의. Firestore와의 데이터 매핑 및 `RULES.md`의 의학적 기준에 따른 '혈당/혈압 상태 진단 로직'이 객체 내부에 통합되어 있습니다.

### 3. 화면 UI (`lib/screens/`)
- **`dashboard_screen.dart`**: 하이브리드 대시보드 메인 화면. 최근 측정한 건강 기록을 바탕으로 혈당과 혈압 상태를 시각적인 카드 형태로 요약 제공하며 주요 기능 네비게이션을 담당합니다.
- **`family_screen.dart`**: 가족 연결 화면. 본인의 연결 코드를 확인하고 가족을 초대하거나, 이미 연결된 가족 목록을 표시하고 관리합니다.
- **`input_screen.dart`**: 건강 기록 등록 화면. 수동 입력과 카메라 기반 OCR 자동 입력을 지원하며, 측정 시점(공복/식전/식후) 선택 및 의학적 피드백 확인 후 Firestore에 데이터를 저장합니다. 식전 기록 시 푸시 알람을 자동 설정합니다.
- **`login_screen.dart`**: 초기 로그인 화면. 카카오 로그인 및 회원가입 진입점 역할을 하며, 인증 후 대시보드로 이동합니다.
- **`my_page_screen.dart`**: 마이페이지 화면. 사용자의 카카오 프로필 정보 표시, 다크모드 전역 설정 토글, 로그아웃 및 데이터 영구 삭제를 동반하는 회원 탈퇴 기능을 제공합니다.
- **`stats_screen.dart`**: 건강 분석 리포트 화면. 기간별(주간/월간 등) 데이터 필터링, `fl_chart` 기반 혈당/혈압 분리 그래프(정상/최고/최저 기준선 및 툴팁) 뷰, 그리고 캘린더 기반 상세 리스트 뷰 및 동적 요약 피드백을 제공합니다.

### 4. 핵심 서비스 (`lib/services/`)
- **`notification_service.dart`**: 로컬 푸시 알림 엔진 (`flutter_local_notifications`). 식후 2시간 뒤 혈당 측정을 잊지 않도록 스마트 푸시 알람 스케줄링을 담당합니다.
- **`ocr_service.dart`**: Google ML Kit 기반 텍스트 인식 엔진. 사진 내에서 숫자를 추출하고, '혈당'/'혈압' 키워드와 숫자의 개수에 따라 지능적으로 필드를 판별하고 매핑하는 OCR 핵심 로직이 구현된 서비스 클래스입니다.

### 5. 공통 위젯 (`lib/widgets/`)
- **`menu_card.dart`**: 대시보드에서 사용하는 카드형 메뉴 버튼 컴포넌트. 최소 터치 영역(60px 이상) 준수 및 터치 시 Scale 작아짐 애니메이션 효과가 적용되어 있어 어르신 친화적 UX를 제공합니다.

### 6. 안드로이드 플랫폼 설정 (`android/`)
- **`app/build.gradle.kts`**: 안드로이드 앱 전용 빌드 설정 파일. 카카오 네이티브 앱 키 설정 및 `flutter_local_notifications` 요구사항을 충족하기 위한 `desugar_jdk_libs` 최신 버전(2.1.5 등) 명시적 관리, 한국어 OCR 모델 의존성 등이 정의되어 있습니다.
